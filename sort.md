Fiche Méthode #sort 
Methode : sort 

1. MA DÉFINITION PERSONNELLE
trie un tableau.

2. PARAMÈTRES ACCEPTÉS
fonction de comparaison 

3. CE QU’ELLE RETOURNE
le meme tableau trie 

4. QUAND L’UTILISER ?
trier des nombre - mettre des donnes dans l'ordre etc ..

5. CE QU’ELLE NE FAIT PAS ?
ne crie pas un nouveau tableau

6. pratique : 
const nums = [10, 2, 5];
nums.sort((a,b)=>a-b);

7. return : 
[2, 5, 10]
